 *defination:- JavaScript is a high-level, interpreted programming language used to make web pages interactive and dynamic.

 #Interpreter:-JavaScript is an interpreted language, which means the browser reads and runs the code directly, line by line, without compiling it first.
                        Speed:Slower due to line-by-line execution
                        Execution:Immediate execution but slower for large program
                        Example in JS:console.log("Hello");   // This will run
                        console.loog("World");  // Error here, execution stops

#Compiler:- A compiler is a special program that translates the entire source code written in a high-level programming language (like C, C++, or Java) into machine code (binary) 
that a computer can understand and execute.
                        Speed:Faster (after compilation)
                        Execution:Longer initially (compile time), but faster later
